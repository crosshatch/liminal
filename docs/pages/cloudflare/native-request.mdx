# NativeRequest

`NativeRequest` is the raw Cloudflare `Request` exposed as an Effect service.

It is provided automatically by `Worker.make(...)` for every incoming request, so you can yield it directly inside your
handler.

## Yield the raw request

```ts
import { Effect } from "effect"
import { NativeRequest } from "liminal"

Effect.gen(function* () {
  const request = yield* NativeRequest

  const url = new URL(request.url)
  const cookie = request.headers.get("cookie")
})
```

## When to use it

Use `HttpServerRequest` for routing, body parsing, and streaming. Reach for `NativeRequest` only when an API requires
the underlying `Request` — for example `Assets.forward`, `ActorRegistry.upgrade(...)`, or third-party libraries that
operate on `Request` directly.

If you build an entrypoint other than `Worker.make(...)`, provide it as `Layer.succeed(NativeRequest, request)`.
